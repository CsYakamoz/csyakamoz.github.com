检查某命令是否存在

来源: How can I check if a program exists from a Bash script?

command -v <the_command>

例子:

if ! command -v COMMAND &> /dev/null
then
    echo "COMMAND could not be found"
    exit
fi

排序版本号

使用 sort 命令, 配合 -V 选项

-V, –version-sort

# 从小到大排序
echo -e '1.2.100.4\n1.2.3.4\n10.1.2.3\n9.1.2.3' | sort -V

# output
# 1.2.3.4
# 1.2.100.4
# 9.1.2.3
# 10.1.2.3

# 从大到小排序
echo -e '1.2.100.4\n1.2.3.4\n10.1.2.3\n9.1.2.3' | sort -rV

# output
# 10.1.2.3
# 9.1.2.3
# 1.2.100.4
# 1.2.3.4

检查某命令(软件)的是满足最低版本

来源: How to compare a program’s version in a shell script?

#!/bin/bash
currentver="$(gcc -dumpversion)"
requiredver="5.0.0"
if [ "$(printf '%s\n' "$requiredver" "$currentver" | sort -V | head -n1)" = "$requiredver" ]; then
       echo "Greater than or equal to ${requiredver}"
else
       echo "Less than ${requiredver}"
fi
